# RFM Customer Segmentation Analysis

## Overview
Customer segmentation model built using BigQuery SQL and Power BI. 287 customers were segmented into 8 behavioral groups based on recency, frequency, and monetary value using a 5-step SQL pipeline with NTILE(10) decile scoring.

## Tools
- BigQuery SQL
- Power BI Desktop
- DAX

## Files
- `rfm_analysis_prject.sql` — Full SQL pipeline (6 steps)
- `rfm_analysis_dashboard.pbix` — Power BI dashboard
- `202501.csv` to `202512.csv` — Raw monthly sales data
